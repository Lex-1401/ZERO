# Firebase Tool

Integrates with the `firebase-tools` CLI.

## Prerequisites

- Install Firebase CLI: `npm install -g firebase-tools`
- Login: `firebase login`

## Actions

### Init

Initialize a Firebase project in the current directory.

```bash
firebase init
```

### Deploy

Deploy your project to Firebase.

```bash
firebase deploy
```

### Serve

Run a local development server.

```bash
firebase serve
```

### List Projects

List your Firebase projects.

```bash
firebase projects:list
```
