# `zero tui`

Abra a UI de terminal conectada ao Gateway.

Relacionado:

- Guia TUI: [TUI](/tui)

## Exemplos

```bash
zero tui
zero tui --url ws://127.0.0.1:18789 --token <token>
zero tui --session main --deliver
```
