# `zero setup`

Inicialize `~/.zero/zero.json` e o workspace de agente.

Relacionado:

- Começando: [Getting started](/start/getting-started)
- Assistente: [Onboarding](/start/onboarding)

## Exemplos

```bash
zero setup
zero setup --workspace ~/zero
```

Para rodar o assistente via setup:

```bash
zero setup --wizard
```
