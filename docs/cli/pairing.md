# `zero pairing`

Aprove ou inspecione requisições de emparelhamento DM (para canais que suportam emparelhamento).

Relacionado:

- Fluxo de emparelhamento: [Pairing](/start/pairing)

## Comandos

```bash
zero pairing list whatsapp
zero pairing approve whatsapp <code> --notify
```
