# `zero skills`

Inspecione habilidades (empacotadas + workspace + overrides gerenciados) e veja o que é elegível vs perde requisitos.

Relacionado:

- Sistema de Skills: [Skills](/tools/skills)
- Config de Skills: [Skills config](/tools/skills-config)
- Instalações ZeroHub: [ZeroHub](/tools/zerohub)

## Comandos

```bash
zero skills list
zero skills list --eligible
zero skills info <nome>
zero skills check
```
