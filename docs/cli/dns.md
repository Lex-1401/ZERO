# `zero dns`

Auxiliares DNS para descoberta wide-area (Tailscale + CoreDNS). Atualmente focado em macOS + Homebrew CoreDNS.

Relacionado:

- Descoberta de Gateway: [Discovery](/gateway/discovery)
- Configuração de descoberta wide-area: [Configuration](/gateway/configuration)

## Configuração

```bash
zero dns setup
zero dns setup --apply
```
