# ADR 002: Estratégia de Squad AIOS + MCP + Skills

## Status

Aceito

## Contexto

O desenvolvimento agêntico requer não apenas um modelo, mas uma infraestrutura de suporte. O AIOS (Agentic Intelligence Operating System) fornece as ferramentas necessárias para que agentes operem com consciência de sistema.

## Decisão

Adotar o ecossistema AIOS + MCP (Model Context Protocol) + Skills para todas as tarefas de desenvolvimento:

- **MCP:** Padronização da comunicação com ferramentas externas.
- **AIOS Squad:** Utilização de agentes especializados (Dev, Sec, QA) em vez de um agente generalista.
- **Skills:** Repositório de capacidades reutilizáveis para automação de tarefas repetitivas.

## Consequências

- **Positivas:** Especialização de tarefas, redução de erros humanos e automação profunda do SDLC.
- **Negativas:** Dependência da infraestrutura AIOS para o fluxo de trabalho completo.
